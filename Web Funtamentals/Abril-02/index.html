Hola J, me gusta tu Query

Recordemos que debemos seguir los siguientes pasos como estructura para usar JQUERY:

  1. Selector como si fuera css => $(".class") o $("#myID")
  2. Buscar el atributo para lo que queremos hacer => $("h1").text()
  3. Si vamos a modificar el valor incluirlo en el parentesis =>("h1").text("VALOR")
